# SmartPet-AI-Log

数智伴宠 (Digital Pet Companion) 🐾
  
“数智伴宠” 是一款以“养宠日志”AI 智能分析为核心功能，辅以多方资源提供的一站式宠物服务平台。
本项目致力于通过 AI 主动式预防与诊断 技术，填补宠物日常健康监测的空白，并构建从“记录-分析-建议-消费”的闭环服务生态。

🌟 核心功能
1. AI 主动式智能日志分析
双重日志系统：支持“私人日志”（用于 AI 健康分析）与“社区分享”（公开社交）。

多宠分类管理：支持如“当当”、“乐乐”等多宠物独立建档，生成个性化分析报告。

症状自动识别：基于 Python 后端 NLP 技术，主动提取日志中的“粪便情况、食欲、精神状态”等关键词。

风险预警：匹配病症数据库，筛选出 2-4 种潜在疾病风险并提供针对性提示。

3. 闭环服务生态

智能商城推荐：根据 AI 报告结果（如“掉毛严重”），优先推荐关联产品（如“化毛膏”）。

精准问诊匹配：若 AI 检测到肠胃炎/过敏等高风险，自动匹配对应擅长领域的兽医专家。

日程提醒：集成疫苗、喂药、驱虫等日常养宠任务提醒。

4. 互动社区

主题化分享：涵盖日常记录、医疗知识、好物分享等 #tag 话题。

多维交互：支持发布内容、评论、点赞及收藏功能。

🏗️ 技术架构
前端：微信小程序原生框架 (WXML / WXSS / JavaScript)
后端：Python (Flask/FastAPI) + NLP 关键词提取算法
数据库：MySQL (用户信息、宠物档案、病症关联数据库)

SmartPet-AI-Log/

🚀 快速开始
前端启动
下载并安装 微信开发者工具。
导入 frontend 文件夹。
修改 app.js 中的服务器地址为你的本地或云端 API 地址。
后端启动
安装依赖：pip install -r requirements.txt
运行服务：python app.py
